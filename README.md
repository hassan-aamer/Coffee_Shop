# Coffee Shop Website ☕

A beautifully designed, responsive static website for a Coffee Shop. This project showcases a modern frontend interface tailored for cafe businesses, featuring essential sections like products, customer reviews, and company history.

## Features ✨

*   **Responsive Design:** Fully optimized for both desktop and mobile devices (includes a dedicated `PhoneResponsive.css`).
*   **Modern UI/UX:** Clean, attractive layout with a welcoming hero section and smooth navigation.
*   **Product Showcase:** A dedicated section to display popular coffee products with prices and "Add to Cart" functionality.
*   **Customer Reviews:** A section highlighting customer testimonials and ratings using Boxicons.
*   **About Us:** A section to share the history and story of the coffee shop.
*   **Interactive Header:** Includes a navigation menu, cart icon, and a search box.
*   **Footer:** Comprehensive footer with social media links, support guides, and contact information.

## Technologies Used 🛠️

*   **HTML5:** For the structure and semantic elements.
*   **CSS3:** For styling, layout, and responsive design (`main.css` & `PhoneResponsive.css`).
*   **JavaScript:** For interactive elements and functionality (`main.js`).
*   **Boxicons:** For scalable vector icons used throughout the site.

## Getting Started 🚀

To run this project locally, simply follow these steps:

1.  **Clone or Download** the repository to your local machine.
2.  Navigate to the project directory: `Coffee_Shop`.
3.  Open the `index.html` file in your preferred web browser. 

*(No server setup or build process is required as this is a static website.)*

## Project Structure 📁

```text
Coffee_Shop/
├── images/            # Contains all images used in the website (logo, products, background, etc.)
├── scripts/
│   └── main.js        # JavaScript file for website interactivity
├── styles/
│   ├── main.css           # Main stylesheet for desktop and general styling
│   └── PhoneResponsive.css # Stylesheet for mobile responsiveness
└── index.html         # The main HTML structure of the website
```
